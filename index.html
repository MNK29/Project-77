Project On Console

Visual Studio is a IDE which helps the developers to develop the program in a easy and simple manner.
1) Download and Install visual studio.
2) Open it and install Live-Server Extension in it.
3) Make necessery changes in the settings of visual studio.

We can open the console screen by the following steps
1) Open a tab and right click somewhere on it.
2) Click on Inspect from the shortcut menu.
3) Click on console.
In the JavaScript file, use console.log() method to print on console.
